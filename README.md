# Beyond-Ocean-Ru

Информация: Перевод квестов в сборке Beyond Ocean (1.1.2) Ссылка на [сборку](https://www.curseforge.com/minecraft/modpacks/beyond-ocean)

Установка: Пример для Prism Launcher:

Скопировать с заменой файлы в корневую директорию модпака. Например, C:\Users\user\AppData\Roaming\PrismLauncher\instances\BeyondOcean\minecraft

После запустите игру и проверьте работу перевода.

Для перевода модов использовал [RTF Extension](https://modrinth.com/resourcepack/rtf-extension)

Если найдёте какие-то ошибки, можете сообщить о них мне в дискорд: narik_20
